## 🎯 Quick Start Guide

### For Investor Demo (15 minutes)

```bash
# 1. Start backend
cd backend
npm run dev

# 2. Start frontend
cd ../frontend
npm run dev

# 3. Open browser
http://localhost:5174

# 4. Navigate to WhatsApp Demo page

# 5. Run through 5 scenarios from DEMO_CONVERSATION_SCENARIOS.md
```

### For Production Deployment (30 minutes)

```bash
# 1. Follow AWS_BEDROCK_SETUP.md to configure AWS

# 2. Update .env
AWS_ACCESS_KEY_ID=your_key
AWS_SECRET_ACCESS_KEY=your_secret
AWS_REGION=us-east-1
MOCK_AI=false

# 3. Deploy enhanced features
cd backend
pm2 restart wazassist-backend

# 4. Monitor logs
pm2 logs wazassist-backend
```
